# NotaryQuickCopy

A simple, professional script organizer for in-person meetings.

NotaryQuickCopy helps you stay organized during client appointments by keeping important talking points, instructions, and repeatable messages in one clean, easy-to-use interface.

Even if you're not a notary, this app works for anyone who needs structured, repeatable communication during meetings.

### 1. Create Folders
Organize your scripts by type, topic, or client.

### 2. Create Files
Each file contains:

- **Read Only Section**  
  A structured script or guide you follow during the meeting.

- **Copy Sections**  
  Text blocks you can instantly copy and paste into emails, forms, or messages.

### 3. Use Favorites
Quickly access commonly used scripts.

### 4. Auto-Save & Locking
- Enable Auto-Save to save automatically.
- Lock files to prevent accidental edits.
- Unsaved changes are clearly shown.
